The Vehicle

Controls have been added to move the vehicle appropriately.
1. Up arrow - move the vehicle forward.
2. Down arrow - move the vehicle in reverse direction.
3. Left arrow - turn the vehicle left (while the up arrow is pressed)
4. Right arrow - turn the vehicle right (while the up arrow is pressed)
5. 5 - Record Mode on, After turning this mode on, we can capture keyframes
6. P - On pressing this keyframes get recorded into a file named keyframes.txt
7. 2 - Play the recorded keyframes
8. 3 - While play mode is on, on pressing this button pictures will get stored into a folder named frames

Linear Interpolation

We have used linear interpolation to generate frames in between keyframes, both for angles and position. For lights we have used the following protocol: If the light is off in one keyframe and on in the next, then in the first half of the frames it is off and in the second half it is on.

Controls
1. 8 - Turns the first light on/off.
2. 9 - Turns the second light on/off.
3. 7 - Turns the headlight on/off.

Linux commands and tools used to create the movie

mogrify -flip -format jpg -quality 100 frame_*.ppm - This command is run in the folder where the images have been stored. It flips the images vertically.
ffmpeg -framerate 60 -i frame_%04d.jpg -codec copy input.mkv - This command creates the movie from the jpg images
ffmpeg -i input.mkv -i final_track.mp3 -c copy -map 0:0 -map 1:0 output2.mkv - This command adds sound track to the movie

Camera

Numpad keys 8,4,6,and 2 have been used to move the camera around the scene.
